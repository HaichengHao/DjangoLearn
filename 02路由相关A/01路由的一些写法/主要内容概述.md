## 主要内容概述  
1. 路由的一些写法
   本小节主要讲的是路由的一些写法,包括如何从apps的指定的app中引入views  
 然后在主路由urls中引入这些路由  
2. 路由的正则写法(旧,了解即可)
- ```python 
  #首先主路由urls.py一定要导入repath
  from django.urls import re_path
    
  urlpatterns = [
        re_path(r'regxtst/(\d+)/',views.regxtst) #important:注意加上r进行转译
  ]     
    ```
  
3. 路由的分发  
 - 在根路由只做分发(结合flask理解就是apps下的__init__.py中只注册蓝图)  
   根路由分发到各个apps中的路由的路由   
 - 首先需要在apps下自己要用的app的文件下新建一个文件urls.py
   
 - ```python  
     #主路由urls.py中这么写
     from django.urls import path,include  
   
     urlpatterns=[
          path('web/',include("apps.web.urls")),
          
     ]
   
   
     #在apps/web/urls.py中这么写
      from django.urls import path
        from .views import home,article
        urlpatterns = [
            path('home/',home),
            path('article/',article)
        ]
     
     
   #然后访问这个http://127.0.0.1:8000/web/home/ 就可以
   ```
